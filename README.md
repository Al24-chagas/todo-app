# 📝 Todo List App

A lightweight Todo List application built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## 🚀 Live Demo

- **AWS S3:** [http://todo-app-alchagas.s3-website.eu-north-1.amazonaws.com](http://todo-app-alchagas.s3-website.eu-north-1.amazonaws.com)
- **GitHub Pages:** [https://SEU-USUARIO.github.io/todo-app/](https://SEU-USUARIO.github.io/todo-app/)

## ✨ Features

- ✅ Add, complete, and delete todos
- 🎯 Priority levels (IBU 0-5) with color coding
- 📂 Categories: Todo, Idea, Tickets to follow
- 🔍 Filter by status or category
- 💾 Data persists in localStorage
- 📱 Responsive design (works on mobile)
- ♿ Accessible (ARIA labels, keyboard navigation)

## 🛠️ Tech Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript (ES6+)
- LocalStorage for persistence

## 📦 Deployment

Hosted as a static site on:
- AWS S3 + Static Website Hosting
- GitHub Pages

## 🖥️ How to Run Locally

Just open `index.html` in any browser — no server needed!
